* [Crab](https://github.com/LinerSu/crab): An abstract domain library for LLVM IR-based program analysis
* [SeaDSA](https://github.com/LinerSu/sea-dsa): A points-to program analysis for LLVM
* [SeaBMC](https://github.com/LinerSu/seahorn): A bounded model checker for LLVM
* [Drift](https://github.com/nyu-acsys/drift): An abstract-interpretation-based static analysis to infer refinement types
